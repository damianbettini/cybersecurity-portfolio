# Hack The Box Writeups

Writeups for **retired** HTB machines only.

## Methodology

Every writeup follows this structure:

1. **Reconnaissance** — Port scanning, service enumeration
2. **Exploitation** — Vulnerability identification and exploitation
3. **Privilege Escalation** — From user to root/admin
4. **Lessons Learned** — What the machine taught me
5. **Defensive Recommendations** — How to protect against these attacks
6. **MITRE ATT&CK Mapping** — Techniques mapped to the framework
